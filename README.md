# Анализ выручки по тарифам подписки (2019–2020)
Анализ продаж подписок на тарифы silver, gold и platinum за 2019–2020 годы.  
Цель — понять динамику выручки, сравнить периоды, выявить лидеров и тренды.
## Навыки и техники
- SQL: LAG, SUM OVER, AVG OVER, RANK, NTILE, CTE
- Агрегация: кумулятивная сумма, скользящее среднее, YoY
- Визуализация: Power BI (дашборды по месяцам и кварталам)
## Ключевые запросы
1. Выручка по тарифу gold за 2020 + изменение к предыдущему месяцу
2. Кумулятивная выручка по тарифам за 1 квартал 2020
3. Скользящее среднее выручки по platinum
4. Сравнение кварталов 2020 с 2019
5. Вклад каждого тарифа в годовую выручку
6. Рейтинг месяцев по количеству продаж
## Ключевые инсайты
- Тариф gold приносит ~40–50% выручки в среднем по году
- В 2020 году наблюдается рост выручки в Q3–Q4 (сезонность?)
- Самый высокий прирост от декабря к январю — +200% у gold
